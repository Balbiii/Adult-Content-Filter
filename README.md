# Adult-Content-Filter
Adult Content Filter lets you hide videos, thumbnails, and content from adult websites based on your own custom keywords and performer names - no built-in lists, no tracking, fully under your control.

## How it works
Add any words or phrases to the Keywords list and the extension will automatically hide any video tile, thumbnail, title, or chat message that contains a match. Add performer names to the Actors list to hide all content featuring those people, even if the keyword doesn't appear in the title. The filter runs in real time and handles dynamically loaded content as pages scroll.

## Features
- Block content by keyword - hide tiles whose title, tag, channel name, or description matches your list
- Block content by performer name - hide any video featuring a specific actor or model, matched by display name or profile URL
- Whitelist specific items - restore a hidden tile and permanently allow it through
- Import and export your settings - back up or transfer your keyword and actor lists as a JSON file

## Privacy
This extension collects no data whatsoever. All settings are stored locally in your browser using the standard extension storage API and never leave your device.
